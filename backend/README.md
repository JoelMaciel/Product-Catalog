# Technologies used

## Back end

- Java
- Spring Boot
- AWS
- Spring Security
- OAuth2 , JWT
- JPA / Hibernate
- Maven

## Deployment in production

- Back end: Heroku , AWS
- Database: Postgresql

# How to run the project

## Back end

Prerequisites: Java 11

```bash
# clone repository
git clone https://github.com/JoelMaciel/Product-Catalog.git

# enter the back end project folder
cd backend

# run the project
./mvnw spring-boot:run
```
